# Python Kiwilytics Project

## 📌 Overview

This project demonstrates practical applications of Python in **data analysis, data engineering, and feature engineering** using real-world datasets.

It focuses not only on visualization, but also on preparing and transforming data for analysis.

---

## 🧰 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

---

## 📊 Project Components

### 🔹 Data Visualization

* Exploratory Data Analysis (EDA)
* Identifying patterns and trends
* Creating visual insights using charts

---

### 🔹 Data Engineering

* Data cleaning (handling missing values, duplicates)
* Data transformation (formatting, type conversion)
* Data integration and structuring
* Preparing raw data for analysis

---

### 🔹 Feature Engineering

* Creating new meaningful features
* Encoding categorical variables
* Aggregations and derived metrics
* Improving data quality for better insights

---

## 📂 Project Files

* `data_visualization.ipynb` → Analysis & visualization
* `data_engineering_issue.ipynb` → Data cleaning & engineering tasks

---

1. Install required libraries:

